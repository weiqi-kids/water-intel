# 新增公司

用戶說「把 XX 加進來」「新增追蹤 XX」「add company」時觸發。

## 步驟

### 1. 研究公司

用 WebSearch 查詢：
- 公司全名、中文名、英文名、別名
- 股票代號（ticker）、交易所（market）、幣別（currency）
- 在供應鏈中的位置：upstream / midstream / downstream
- 角色描述（如：（依產業填寫，如：設備、材料、製造））
- IR 頁面 URL
- 新聞頁面 URL 或 RSS feed

### 2. 更新 configs/companies.yml

```yaml
- id: new_company
  name: New Company
  aliases:
  - 新公司
  - NEWCO
  ticker: NEWCO
  market: NASDAQ
  currency: USD
  position: midstream
  role: 角色描述
  description: 業務描述
  ir_url: https://ir.newcompany.com/
  news_url: https://newcompany.com/news/
  ir_rss_url: https://ir.newcompany.com/rss/news-releases.xml  # 如果有
  fetch_mode: rss  # rss / http / playwright
  upstream:
  - supplier_id
  downstream:
  - customer_id
```

### 4. 註冊到 fetch_companies.py

在 `scripts/fetch_companies.py` 的 `FETCHERS` dict 中加入：

```python
from fetchers.new_company import NewCompanyFetcher
# ...
FETCHERS = {
    # ... existing
    "new_company": NewCompanyFetcher,
}
```

### 5. 重新 normalize

```bash
python scripts/normalize.py
cp data/normalized/companies.json site/data/
```

### 6. 驗證

```bash
git add configs/companies.yml fetchers/new_company.py scripts/fetch_companies.py data/normalized/ site/data/companies.json
git commit -m "feat: add {company_name} to tracking"
git push origin main
gh workflow run "Daily Data Ingest" -R $(gh repo view --json nameWithOwner -q .nameWithOwner)
```

等 pipeline 完成，確認新公司出現在：
- 供應鏈圖節點
- 股價走勢圖（次日開始有資料）
- 事件時間軸（如果當天有新聞）

## 注意

- `fetch_mode` 和 `ir_rss_url` 在 `companies.yml` 和 `fetchers/*.py` 兩邊都要設定
- 新公司的股價歷史會從加入日開始累積（`fetch_stocks.py` 自動 backfill）
- 財務資料需等下次 `fetch_financials.py` 執行才有
