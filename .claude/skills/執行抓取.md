# 執行抓取

用戶說「手動跑一次」「重新抓取」「run pipeline」時觸發。

## 方式一：GitHub Actions（推薦）

```bash
gh workflow run "Daily Data Ingest" -R $(gh repo view --json nameWithOwner -q .nameWithOwner)
```

等待完成（約 5-7 分鐘）：
```bash
sleep 15
RUN_ID=$(gh run list -R $(gh repo view --json nameWithOwner -q .nameWithOwner) --workflow="Daily Data Ingest" --limit 1 --json databaseId -q '.[0].databaseId')
echo "Run ID: $RUN_ID"
# 等待完成
gh run watch $RUN_ID -R $(gh repo view --json nameWithOwner -q .nameWithOwner)
# 檢查結果
gh run view $RUN_ID -R $(gh repo view --json nameWithOwner -q .nameWithOwner)
```

檢查錯誤：
```bash
JOB_ID=$(gh run view $RUN_ID -R $(gh repo view --json nameWithOwner -q .nameWithOwner) --json jobs -q '.jobs[0].databaseId')
gh run view --job=$JOB_ID -R $(gh repo view --json nameWithOwner -q .nameWithOwner) --log 2>/dev/null | grep "ERROR:fetchers" | sed 's/.*ERROR:fetchers[^:]*://' | sort
```

## 方式二：本地執行（除錯用）

```bash
source .venv/bin/activate

# 按順序執行
python scripts/fetch_stocks.py
python scripts/fetch_companies.py
python scripts/fetch_rss.py
python scripts/normalize.py
python scripts/fetch_financials.py
python scripts/fetch_holders.py
python scripts/generate_cards.py

TODAY=$(date +%Y-%m-%d)
python scripts/enrich_event.py --date $TODAY --input "data/raw/$TODAY/companies.jsonl"
python scripts/enrich_event.py --date $TODAY --input "data/raw/$TODAY/rss.jsonl"
python scripts/generate_metrics.py --date $TODAY
python scripts/generate_daily.py --date $TODAY
python scripts/generate_7d_report.py --date $TODAY
python scripts/generate_llm_analysis.py --date $TODAY
python scripts/sync_to_frontend.py
python scripts/generate_financials_history.py
python scripts/generate_config_stats.py

# 複製到 site
cp data/normalized/stocks.json site/data/
cp data/normalized/companies.json site/data/
cp reports/daily/$TODAY.json site/data/reports/daily/
cp reports/7d/$TODAY.json site/data/reports/7d/
```

## Pipeline 順序不可調

步驟有先後依賴關係，不可跳過或亂序：
1. fetch → 2. normalize → 3. enrich → 4. metrics → 5. reports → 6. sync → 7. history

## 完成後拉取最新

如果是在 GitHub Actions 上跑的：
```bash
git pull origin main
```
