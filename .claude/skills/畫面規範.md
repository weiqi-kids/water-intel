# 畫面規範

修改 `site/index.html` 前必須讀取此 skill。

## Design Tokens（CSS Variables）

### 顏色使用規則

**文字顏色** — 禁止使用硬寫 hex，必須用 var：

| 用途 | CSS Variable | 禁止用的 hex |
|------|-------------|-------------|
| 主要文字 | `var(--text-primary)` | `#1e2030`, `#333`, `#222` |
| 次要文字 | `var(--text-secondary)` | `#555`, `#666` |
| 輔助文字 | `var(--text-muted)` | `#888`, `#999`, `#aaa` |

**語意色** — 表達狀態意義：

| 用途 | CSS Variable | Hex fallback（僅 JS/SVG） |
|------|-------------|--------------------------|
| 負面/下跌/危險 | `var(--color-critical)` | `var(--hex-critical, #c93135)` |
| 警示 | `var(--color-warn)` | `var(--hex-warn, #8a7020)` |
| 正面/上漲/通過 | `var(--color-pass)` | `var(--hex-pass, #1e8050)` |
| 資訊 | `var(--color-info)` | `var(--hex-info, #2a6bb8)` |
| 品牌/互動 | `var(--color-brand)` | `var(--hex-brand, #3b82f6)` |

**背景色：**

| 用途 | CSS Variable |
|------|-------------|
| 頁面底色 | `var(--bg-base)` |
| 卡片/面板 | `var(--bg-surface)` |
| hover 狀態 | `var(--bg-hover)` |
| 淡化區域 | `var(--bg-muted)` |

**供應鏈位置色：**

| 位置 | CSS Variable | Hex（JS 用） |
|------|-------------|-------------|
| 上游 | `var(--color-upstream)` | `var(--hex-upstream, #b86a2a)` |
| 中游 | `var(--color-midstream)` | `var(--hex-midstream, #2a6bb8)` |
| 下游 | `var(--color-downstream)` | `var(--hex-downstream, #1e8050)` |

### JS/SVG 中的顏色

在 JavaScript 動態生成的 HTML 或 D3 SVG 中，**必須使用 hex fallback 格式**：

```javascript
// 正確
style="color:var(--hex-pass, #1e8050)"
.attr('fill', 'var(--hex-critical, #c93135)')

// 錯誤
style="color:#1e8050"
.attr('fill', '#c93135')
```

### 字體

```css
font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif;
```

不可自行指定其他字體。

### 字級規範

| 用途 | 大小 |
|------|------|
| 頁面標題 | `1.5rem` |
| 面板標題 | `0.85rem` (font-weight: 600) |
| 正文 | `0.8rem` |
| 輔助/標籤 | `0.75rem` |
| 極小（sparkline 標籤等） | `0.7rem` |

### 間距

| 用途 | 大小 |
|------|------|
| 面板間距 | `20px` (grid gap) |
| 面板內 padding | `15px` |
| 元素間 margin | `8px` / `12px` |

### 邊框

```css
border: 1px solid var(--border-subtle);
border-radius: 8px;  /* 面板 */
border-radius: 4px;  /* 按鈕、tag */
```

## 修改前驗證 Checklist

修改 `site/index.html` 後，執行以下檢查：

### 1. 檢查有無新增硬寫 hex

```bash
git diff site/index.html | grep "^+" | grep -E "#[0-9a-fA-F]{3,6}" | grep -v "var(--" | grep -v "^+++" | grep -v "hex-"
```

如果有輸出，代表新增了硬寫 hex — 必須改為 CSS variable。

### 2. 檢查字體大小

```bash
git diff site/index.html | grep "^+" | grep "font-size" | grep -v "0\.\(7\|75\|78\|8\|85\|9\)rem\|1\.5rem\|1rem\|0\.65rem"
```

非標準字級需要有明確理由。

### 3. 視覺確認

```bash
python3 -m http.server 6229 -d site
# 開瀏覽器確認：
# - 配色一致（沒有突兀的顏色）
# - 字體大小層級正確
# - hover 效果正常
# - 響應式（縮小視窗檢查）
```
